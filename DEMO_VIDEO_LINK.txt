===========================================
LOCAL DEVELOPMENT
===========================================

To run the example locally:

1. Navigate to example directory:
   cd examples/rideshare

2. Install dependencies:
   npm install

3. Start development server:
   npm run dev

4. Visit in browser:
   http://localhost:3000
